# 라멘 도장깨기

여러 사용자가 각자의 라멘집 방문과 가고 싶은 장소를 기록하는 도장깨기 서비스의 도메인 언어다.

## Language

**라멘집**:
방문하거나 방문하고 싶은 라멘 판매 장소. 지도, 방문 기록, 위시리스트의 중심이 되는 장소 단위다.
_Avoid_: 가게, 식당, place

**사용자**:
서비스 안에서 자기 방문 기록과 위시리스트를 소유하는 사람. 로그인 도입 후 방문 기록과 위시리스트의 소유 기준이 된다.
_Avoid_: 계정, 멤버, account

**방문 기록**:
특정 라멘집에서 특정 날짜에 먹은 메뉴와 맛 평가를 남긴 기록. 한 라멘집은 여러 방문 기록을 가질 수 있다.
_Avoid_: 리뷰, 일기, history

**위시리스트**:
아직 가보지 않았거나 다시 가고 싶은 라멘집을 따로 표시한 목록 항목. 라멘집 자체와는 별도의 사용자 의도다.
_Avoid_: 북마크, 찜

**태그**:
라멘집을 스타일이나 특징으로 분류하는 짧은 이름. 예: 쇼유, 시오, 이에케이, 츠케멘.
_Avoid_: 카테고리, 라벨

**도장깨기**:
사용자별 라멘집 방문 기록이 쌓이며 진행되는 방문 진행 상태. MVP에서는 통계 테이블 없이 방문 기록에서 계산한다.
_Avoid_: achievement, badge
